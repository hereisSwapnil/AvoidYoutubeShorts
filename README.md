# AvoidYoutubeShorts

A Chrome extension that blocks YouTube Shorts to help you stay focused on meaningful content.

## Features

- Automatically hides YouTube Shorts content
- Statistics dashboard to track viewing habits
- Pause/resume functionality
- Privacy-first: all data stored locally

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/hereisSwapnil/AvoidYoutubeShorts.git
   cd AvoidYoutubeShorts
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Build the extension:
   ```bash
   npm run build
   ```

4. Load in Chrome:
   - Go to `chrome://extensions/`
   - Enable "Developer Mode"
   - Click "Load unpacked" and select the `dist-prod/` folder

## Usage

The extension works automatically once installed. Click the extension icon to view statistics or pause/resume blocking.
